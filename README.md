## 关于这份日报

我们的大脑是一台预测机器，它用过去的经验生成对世界的"最佳猜测"。这意味着我们时刻活在一种受控的幻觉中，舒适区就是我们预测误差最小的区域。

破界日报的目标是主动制造"预测误差"——每天带你进入一个你不熟悉的领域，让你的大脑被迫进入学习模式。
